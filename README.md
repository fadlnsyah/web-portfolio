## ğŸš€ Quick Start

### Prerequisites

- **Node.js** >= 18.x
- **npm** or **yarn**

### Installation

1. **Clone the repository**

   ```bash
   git clone https://github.com/fadlnsyah/web-porto.git
   cd web-porto
   ```

2. **Install dependencies**

   ```bash
   npm install
   # or
   yarn install
   ```

3. **Run development server**

   ```bash
   npm run dev
   # or
   yarn dev
   ```

4. **Open in browser**
   ```
   http://localhost:3000
   ```

## ğŸ¨ Theming

Website sudah dilengkapi dengan **Dark Mode** dan **Light Mode** yang dapat di-toggle melalui navbar.

Untuk kustomisasi tema, edit file `tailwind.config.js`:

```javascript
module.exports = {
  darkMode: "class",
  theme: {
    extend: {
      colors: {
        // Kustomisasi warna di sini
      },
    },
  },
};
```
